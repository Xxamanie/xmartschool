GitHub Secrets Configuration
============================

Add these secrets to your GitHub repository:
Settings → Secrets and variables → Actions → New repository secret

1. SECRET NAME: FIREBASE_SERVICE_ACCOUNT_XMART_SCHOOL
   VALUE: Your Firebase service account JSON (entire file as one line)
   HOW TO GET:
   - Go to Firebase Console (console.firebase.google.com)
   - Select project: xmart-school
   - Settings → Service Accounts
   - Click "Generate new private key"
   - Copy the entire JSON content and paste here

2. SECRET NAME: VITE_API_BASE_URL
   VALUE: https://xmartschool.onrender.com/api
   NOTE: Replace "xmartschool" with your actual Render service name if different

3. SECRET NAME: VITE_GEMINI_API_KEY (OPTIONAL)
   VALUE: Your Gemini API key (if using Gemini features)
   NOTE: Can be left empty if not used

VERIFY SECRETS ARE SET:
- Go to repository Settings
- Click Secrets and variables → Actions
- You should see all 3 secrets listed (values are hidden)
- Green checkmarks indicate they're set correctly

IMPORTANT:
- Do NOT commit secrets to git
- GitHub Actions will use these secrets automatically
- Frontend builds will use VITE_* secrets
- Backend uses DATABASE_URL from Render environment settings (not GitHub secrets)
